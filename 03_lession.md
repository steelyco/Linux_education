# Работа с жесткими дисками в Linux

## Основные разделы Linux
 - **/** – корневая файловая система, самый большой раздел
- /**boot** – загрузочный раздел. На нем хранятся функции и библиотеки ядра, образ стартовой корневой системы, образ ядра системы
- /**home** – домашние папки
- /**root** – домашняя папка суперпользователя
- /**etc** – конфигурация системы и ее компонентов
- /**opt** – папка для ПО от третьих поставщиков
- /**var** – часто изменяемые данные
- /**usr**– все установленные пакеты программ, документация, исходный код ядра
- /**tmp** – временные файлы;_
- **swap** – раздел подкачки, никуда не монтируется.

На каждом жестком диске моно создать не более **4** разделов. Если понадобится больше, то можно создать расширенный раздел, который может содержать множество логических разделов.

Жесткие диски именуются по порядку их подключения - *sda*, *sdb*, *sdc*
Разделы на жестком диске именуются по порядку - *sda1*, *sda2*. При этом первые **4** зарезервированы под основные разделы, логические разделы начинают отсчет с **5**.

Для создания и прочих действий с разделами используется утилита **fdisk**
Для создания файловой системы используется утилита **mkfs**
Подключать созданные разделы можно вручную при помощи команды **mount** имя_раздела пустой_каталог, можно автоматически с использованием файла настроек **/etc/fstab**

**fdisk -l** - Показывает существующие диски и разделы
**fdisk /dev/sda** - Заходит внутрь диска и можно менять его разделы и тд.
**mount** имя_раздела пустой_каталог - Монтирует раздел диска в указанный каталог 
**umount** имя_раздела указанный_каталог - отмонтирует указанный каталог 

## LVM

LVM (Linux Volume Manager) - менеджер для управления томами в Linux. Позволяет создать поверх физических разделов логические тома. Можно добавить несколько физических разделов в одну логическую группу томов (volume group 1). После объединения в одну группу томов можно создавать логические тома не указывая где они находятся.

![](Pasted%20image%2020260115150703.png)

*pvdisplay* - Показывает физические тома (Physical volume) - разделы физического диска
*vgdisplay* - Показывает группу томов (Volume group) - объединённая группа из нескольких физических разделов диска
*lvdisplay* - Показывает логические тома которые есть в системе

Для создания Логических тома в LVM надо:
	1. Создать раздел или разделы на диске или дисках с помощью *fdisk*
	2. Создать Физические Тома pvcreate имя_раздела
	3. Создать Группу Томов *vgcreate* имя_разделов
	4. Создать Логический Том *lvcreate* -n имя_логического_тома -L 5G имя_логического_тома
	5. Нужно отформатировать разделы логических томов - *mkfs.ext4* путь_логического_тома
