# Управление библиотеками и пакетами

## Управление библиотеками

Библиотеки - это набор функций, используемый ПО при работе. Библиотеки могут входит в состав программного обеспечения, а могут хранится отдельно, загружаясь в оперативную память по мере необходимости. Обычно процесс установки библиотеки - процесс установки пакета ПО.

В Linux библиотеки хранятся в папках:
- /**lib**
- /**usr**/**lib**
- прочие пути, указанные в /**etc**/**ld.so.conf**

Для определения используемых библиотек программой используется команда **ldd** *путь_к_программе*

## Управление пакетами Debian/Ubuntu

Вообще для работы с пакетами в Debian подобных системах присутствует утилита **dpkg**, являющаяся громоздким инструментом автоматизации работы с пакетами, поэтому больше популярностью при работе с пакетами используют **apt**. 

**APT** умеет автоматически разрешать зависимости и работать с сетевыми репозиториями, в отличии от **dpkg**. 

Команды для использования **apt**:
- **apt update** - обновить информацию о репозиториях
- **apt upgrade** - обновить пакеты
- **apt install** - устанавливает пакет
- **apt remove** - удаляет пакет
- **apt search** - найти пакет
- **apt depends** - показать зависимости этого пакета
- **apt rdepends** - показать что зависит от пакета
- **apt show** - показать информацию о пакете

Перечень репозиториев apt лежат в файле /etc/apt/sources.list
Настройки менеджера пакетов apt лежат в файле /etc/apt/apt.conf
