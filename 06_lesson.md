# Работа в командной строке Linux

Командная строка - отдельная программируемая среда со своими настройками, возможностями и инструментами. 
В основном используется оболочка bash. Эту оболочку можно изменять под себя.
Для изменения настроек оболочки пользователя можно изменить файл .bashrc


Для работы в консоли используются команды:
**sudo** -*i* - переключение и создание root сессии
**cat** /*etc*/*passwd* - можно посмотреть какая оболочка используется пользователем 
**cd** - перейти в каталог
**ls** - вывод содержимого каталога
**echo** - вывод текста в консоль
**touch** - обновление времени редактирования файла ил создание нового файла 
*uname* - вывод имени ОС

Консоль как отдельная рабочая среда содержит свои переменные: переменные среды (глобальные переменные в ОС) и обычные переменные (работают в пределах запущенного сеанса). Для просмотра переменных используют следующие команды:
**env** - посмотреть все переменные среды 
**pwd** - посмотреть текущую директорию
